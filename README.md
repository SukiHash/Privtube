# PrivTube 📺

PrivTube is a simple web application designed to display YouTube videos based on search queries. It provides a clean and private viewing experience kinda like oyutube

## Features ✨

- **Search YouTube**: 🔍 Enter keywords to search for videos on YouTube.
- **Infinite Scroll**: 🔄 Automatically loads more videos when you scroll to the bottom of the page.
- **Private Usage**: 📜 None Of your data is saved and used for selling, mining etc,).
- **Responsive Design**: 📱 Optimized for various screen sizes.
